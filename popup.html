Open resources

Acknowledgements

YomiSaver is made possible by open-source tools, APIs and community-maintained language resources.

  • Kuromoji.js Japanese tokenisation and furigana processing. Visit project
  • Jisho.org API Readings, definitions and dictionary data. Visit Jisho
  • JMdict Japanese-English dictionary data maintained by the EDRDG. Visit project
  • Tanos JLPT Resources Community JLPT vocabulary reference data credited to Jonathan Waller. Visit resource
  • Bluskyo/JLPT_Vocabulary Cleaned JSON and CSV conversions of JLPT vocabulary data. Visit project
  • Chrome Extensions Platform Browser APIs used to deliver YomiSaver. View documentation
